通过后缀表达式计算逻辑表达式的一个demo

发表于 2019-06-06 更新于 2019-08-08 分类于 算法 阅读次数： Valine：

有一个项目上的需求，需要计算逻辑表达式的结果，比如 (a&b)|(a|b)，似曾相识的感觉，当初学习数据结构时就学了很多通过栈或者树来实现计算表达式结果(面试时二面问的算法也是括号匹配问题，和这个类似)，这次终于派上用场了，参考了下一个大佬写的代码（https://www.cnblogs.com/flyingbread/archive/2007/02/03/638932.html）原文是计算算术表达式，这里做了一些修改

思路

  1. 语言上改用java实现
  2. 由算术表达式计算变成了逻辑表达式计算
  3. 中缀表达式转换成后缀表达式、计算表达式结果这两个步骤改成使用两个栈来存储结果应对可能出现的异常

大体上的逻辑遵守了原作者的思路：

  1. 将中缀表达式转换成后缀表达式
  2. 通过计算后缀表达式来计算结果

输入

(a&b)|(a|b) （其中a=false，b=true，ab的真值获取在标记有todo的那一行）

输出

true
